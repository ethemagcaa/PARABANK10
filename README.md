ParaBank Web Sitesi Team 10 Test Çalışması

Proje Amacı :
https://parabank.parasoft.com/parabank/index.htm web sitesi için belirlenen 8 adet user story nin test edilmesi
amaçlanmaktadır. Böylece sitenin kullanılabilirliğini ve stabil çalışıp çalışmadığını
görebileceğiz

Proje Hedefleri:
  US_01_Reg Kullanıcı Kaydı Oluşturma (Register)
  US_02_Login Kullanıcı Girişi Yapabilme (Login)
  US04_NewAcc  Kullanici olarak yeni bir banka hesabi acma
  US05_AccountOverview Hesap detaylarini goruntuleme
  US07_UpdateContactInfo Iletisim bilgileri guncelleme
  US08_Loan Kullanıcı olarak kredi basvurusu yapma

Proje Teknoloji ve Araçları :
Programming Language : Java JDK 1.8 se
Test Automation Tool : cucumber:7.9.0-testng:7.9.0
Version Control : Git & GitHub
Project Management : Jira & XRay

Kurulum  :
GitHub'da merkez proje oluşturulup master branch koruma altına alındı.
Tüm takım üyeleri İntellij üzerinden projeyi vcs ile klonlayıp kendi branchlarını oluşturdu.
Jirada yeni proje oluşturulup product backlog oluşturuldu.
Tüm test caseler intellij üzerinden çalıştırılıp testler tamamlandı.
Projenin baştan sona kadar yönetimi için Jira ve Xray kullanılmıştır.
